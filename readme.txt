WordPress plugin to extend default search to include meta fields

== Description ==

This WordPress plugin extends the default search to include meta (custom) fields

Major features in WP Search Include Meta Fields include:

* No configuration necessary
* Integrates with core search functionality
* Doesn't alter the database

== Installation ==

Upload the "wp-search-include-meta-fields" folder to your wp-content/plugins directory, then Activate it.
